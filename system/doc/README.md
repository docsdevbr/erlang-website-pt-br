# Documentação do Sistema Erlang/OTP

A documentação do sistema Erlang/OTP é uma coleção de guias que descrevem como
usar o Erlang/OTP e diferentes aspectos do trabalho com o Erlang/OTP.
Os guias são:

- [Guia de instalação](installation_guide/installation_guide.md) - Descreve como
  compilar e instalar o Erlang/OTP no Unix e no Windows.
- [Primeiros passos com o Erlang](getting_started/getting_started.md) - Descreve
  como começar a programar em Erlang.
- [Princípios do sistema](system_principles/system_principles.md) - Descreve
  como compilar sistemas Erlang/OTP.
- [Princípios de design do OTP](design_principles/design_principles.md) -
  Descreve como compilar aplicações Erlang/OTP.
- [Exemplos de programação](programming_examples/programming_examples.md) -
  Exemplos de uso de registros, funções, compreensões de lista e sintaxe de
  bits.
- [Manual de referência do Erlang](reference_manual/reference_manual.md) - Esta
  seção é o manual de referência do Erlang.
  Ela descreve a linguagem de programação Erlang.
- [Guia de eficiência](efficiency_guide/efficiency_guide.md) - Descreve como
  escrever código eficiente em Erlang.
- [Tutorial de interoperabilidade](tutorial/tutorial.md) - Esta seção informa
  sobre interoperabilidade, ou seja, troca de informações, entre o Erlang e
  outras linguagens de programação.
  Os exemplos incluídos tratam principalmente da interoperabilidade entre Erlang
  e C.
- [Guia da pessoa usuária de sistemas embarcados](embedded/embedded.md) - Esta
  seção descreve as questões específicas da execução do Erlang em um sistema
  embarcado.
- [Declarações VEX](vex/vulnerabilities.md) - Esta seção descreve como o
  Erlang/OTP reporta declarações OpenVex e seus significados para terceiros.
